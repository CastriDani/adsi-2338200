HTML5

HTML (Hypertext Markup Language) is the most basic component of the Web. Define the meaning and structure of web content. In addition to HTML, other technologies are generally used to describe the appearance/presentation of a web page (CSS) or the functionality/behavior (JavaScript).

CSS

CSS stands for Cascading Style Sheets. Basically, it is a language that manages the design and presentation of web pages, that is, how they look when a user visits them. It works together with the HTML language that takes care of the basic content of the pages. They are called "cascading" style sheets because you can have multiple sheets and one of them with properties inherited properties.

JavaScript

JavaScript is a programming or scripting language that allows you to implement complex functions on web pages, every time a web page does more than just sit there and display static information for you to see, it displays timely content updates, interactive maps , 2D/3D graphics animation, scrolling video player machines, etc.

FIGMA

Figma is a primarily web-based vector graphics editor and prototyping tool, with additional offline features enabled by desktop applications on macOS and Windows.
